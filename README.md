# HR Analytics Dashboard

## Overview

HR Analytics Dashboard is a Power BI project designed to analyze employee attrition and workforce data. The dashboard helps organizations understand employee trends, identify key factors affecting attrition, and make data-driven HR decisions.

## Objectives

- Analyze employee attrition patterns
- Monitor workforce demographics
- Identify high-risk employee segments
- Track department-wise employee performance
- Support HR decision-making through data visualization

## Features

- Employee Attrition Analysis
- Department-wise Insights
- Gender Distribution Analysis
- Age Group Analysis
- Job Role Analysis
- Education Field Analysis
- Interactive Filters and Slicers
- KPI Cards for Quick Insights

## Tools and Technologies Used

- Power BI
- Microsoft Excel
- Data Visualization
- Data Cleaning
- Data Analysis

## Dashboard Insights

The dashboard provides insights into:

- Overall Attrition Rate
- Total Employees
- Active Employees
- Average Employee Age
- Attrition by Department
- Attrition by Job Role
- Attrition by Education Field
- Attrition by Gender
- Attrition by Age Group
